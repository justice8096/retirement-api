## 3. Implicit / Metadata Labeling Requirements

### 3.1 China Implicit Label Specification

All AI-generated content must embed metadata including:

| Metadata Field | Description | Format |
|---------------|-------------|--------|
| Provider name | Name of the AI content provider | String |
| Content attributes | Type of content, generation method | Structured tags |
| Content number | Unique identifier for the content piece | Alphanumeric ID |
| Generation timestamp | When the content was generated | ISO 8601 |

**Implementation notes:**
- Metadata must be embedded in the file itself (not just a database record)
- Platforms distributing AI content must verify implicit labels exist
- Platforms must add explicit labels if they detect AI content lacking them
- Users may request removal of explicit labels, but implicit labels must remain
- Logs of label modifications must be retained for 6+ months

- [ ] Explicit labels rendered on all AI-generated content
- [ ] Metadata embedded in output files (images: EXIF/XMP; audio: ID3/metadata; video: container metadata; text: document properties)
- [ ] Watermark encoding implemented
- [ ] Detection tool/API built and documented
